Ruru's Garden Inspo API Documentation

This API allows you to manage your database of plants.

Endpoints

  • GET /plants - Get all plants from the database.

    • - Will return a 200 code if successful


  • GET /plants/:id - Get one plant with the specified ID.

    • - Will return a 200 code if successful


  • POST /plants - Create a plant.

    • - Common name is required

    - Sun exposure is required

    - Zone is required

    - Will return a 201 code if successful


  • DELETE /plants/:id - Delete a plant with the specified id.

    • - Will return a 204 code if successful


  • PUT /plants/:id - Update a plant with the specified id.

    • - Common name is required

    - Sun exposure is required

    - Zone is required

    - Will return a 200 code if successful
